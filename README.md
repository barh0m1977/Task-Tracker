
# Task Tracker iOS App

A simple and modern Task Tracker iOS application built using SwiftUI, designed to help users manage daily tasks efficiently with a clean and minimal interface.

## Overview

Task Tracker allows users to create, complete, and track tasks in real time. The app supports task completion states with visual feedback such as strikethrough text and color changes.

The UI is optimized for iOS with a clean layout, simple interactions, and fast performance.
